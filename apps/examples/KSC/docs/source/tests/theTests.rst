Utils
------

A collection of several auxiliary scripts that are used to KSC application input
parameter estimation, visualisation of the clustering and hyper parameter tuning 
results or performing a chain of actions (e.g. data generation, invoking the KSC 
application or generate shell scripts for that, etc..) needed for some of the 
tests.
